Personalize Your Event Code

Set Your Event Location

Guest can only use your event code to get to and/or from your event location(s). You can set up to 5 event venues on a single code.

Schedule Your Event

Your event code will only be usable during the dates and times you set. You can create an event code that is valid for as short as a few hours or up to 7 days.

Customize Your Budget

Set the number of attendees and the dollar amount you want to designate per person. Once your code expires, you’ll be refunded for the unused amount after 48 hours.
